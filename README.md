# Year in Books

## Setup

```bash
npm i
npm start
```

## Render

```bash
npx remotion render
```

## Share on Instagram

- Render in Remotion.
- Upload the video to IG on desktop.
- After selecting the file, click the little `<>` icon in the lower left corner
  of the video and then select the "Original" aspect ratio. If you don't, IG
  will crop the video.
- Complete.
